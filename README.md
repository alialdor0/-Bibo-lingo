# Bibo Lingo

تطبيق تعلم اللغة الإنجليزية بأسلوب قصصي سينمائي.

## هيكل المشروع

```
BiboLingo/
├── App.js                    # نقطة الدخول (11 سطر)
├── package.json              # التبعيات
├── app.json                  # إعدادات Expo
├── babel.config.js           # إعدادات Babel
├── data/
│   └── index.js              # كل بيانات التطبيق (352 سطر)
├── context/
│   └── AppContext.js         # إدارة الحالة العامة (81 سطر)
├── components/
│   ├── WheelPicker.js        # مكون القائمة الدوارة (143 سطر)
│   └── BiboCard.js           # مكونات مشتركة (84 سطر)
├── navigation/
│   └── AppNavigator.js       # نظام التنقل (50 سطر)
├── screens/
│   ├── Login.js              # تسجيل الدخول (84 سطر)
│   ├── Onboarding.js         # الإعداد والبطاقة التعريفية (417 سطر)
│   ├── TrackSelect.js        # اختيار المسار (132 سطر)
│   ├── Main.js               # الرئيسية + الملف + الإعدادات (347 سطر)
│   ├── Story.js              # القصة بـ 7 مراحل (478 سطر)
│   ├── Dict.js               # القاموس مع تمارين (332 سطر)
│   ├── Rescue.js             # إنقاذ الكلمات (224 سطر)
│   ├── Leaderboard.js        # لوحة المتصدرين (124 سطر)
│   ├── Coop.js               # التعاون (441 سطر)
│   └── Store.js              # متجر القرطاسية (251 سطر)
└── assets/
    └── (صور بيبو والأصوات لاحقاً)
```

## تشغيل المشروع

```bash
npm install
npx expo start
```

## الميزات

- 5 مسارات درامية: Spy, Love, Family, Crime, Medical
- 7 مراحل تعليمية لكل كلمة
- نطق صوتي باستخدام expo-speech
- نظام القرطاسية (أقلام، ممحاة، أوراق)
- متجر جواهر مع هدايا إعلانية
- تعاون بين مستخدمين
- لوحة متصدرين
- دعم العربية والإنجليزية
